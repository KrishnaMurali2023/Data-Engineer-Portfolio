# Data Engineer Portfolio Website
A modern, responsive **Data Engineer Portfolio** built using React + Vite + Tailwind CSS.  
It showcases real-world Data Engineering skills, projects, and experience.

# What I Built
This portfolio demonstrates:
- Real-time streaming pipeline projects (Kafka + Spark)
- ETL automation workflows (Airflow)
- Cloud data lake architecture (AWS S3 + Glue + Athena)
- Interactive project and experience modals
- Responsive UI design for all devices

# Tech Stack
- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)

# Portfolio Sections
- Hero section with introduction
- Skills (Data Engineering stack)
- Projects with detailed popups
- Experience timeline
- Contact form UI

# Highlights
- Fully responsive design
- Interactive UI components
- Clean modern dark theme
- Modular React components
